# Personal Portfolio Website

This project is a personal portfolio website created as part of Assignment 3,
building on the work done in Assignments 1 and 2.
It extends the website with advanced features including API integration, 
project filtering and sorting, state management, and performance optimizations.

---

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub REST API

---

## How to Run the Project
1. Download or clone the repository.
2. Open the `index.html` file in any web browser.
3. The website will load locally without the need for any additional setup.

> Note: The GitHub Repositories section requires an internet connection to fetch live data from the GitHub API.

---

## Features
- **About Me section** introducing background and goals
- **GitHub Repositories section** that dynamically fetches and displays public repositories using the GitHub REST API
- **Projects section** with images and expandable project descriptions
- **Project filtering** by category (All, UI/UX, Game)
- **Project sorting** by scale (Large → Small or Small → Large)
- **Contact form** with validation and feedback messages
- **Navigation menu** for quickly moving between sections
- **Responsive layout** that adapts to different screen sizes
- **Greeting message** that combines the time of day with the visitor's saved name
- **Theme toggle button** allowing users to switch between light and dark mode, remembered across visits
- **Visitor name storage** that saves and displays the visitor's name using localStorage
- **Performance optimizations** including image compression, lazy loading, and image preloading
- **SEO improvements** including a meta description tag
